# screens

O diretório `screens` contém os componentes de tela da aplicação, cada um responsável por uma funcionalidade específica. As telas são organizadas em subdiretórios, onde cada subdiretório pode conter um ou mais arquivos relacionados à tela correspondente.

## Estrutura do Diretório

- **cadastrarAlimento**: Tela para cadastrar novos alimentos.
- **FAQs**: Tela que apresenta perguntas frequentes.
- **cadastro**: Tela de cadastro de usuários.
- **editProfile**: Tela para edição do perfil do usuário.
- **home**: Tela inicial da aplicação.
- **listAlimentos**: Tela que lista todos os alimentos disponíveis.
- **selecao**: Tela para seleção de opções.
- **questionario**: Tela que apresenta um questionário ao usuário.
- **profile**: Tela que exibe o perfil do usuário.
- **userDietas**: Tela que mostra as dietas do usuário.
- **userAlimentos**: Tela que lista os alimentos do usuário.
- **login**: Tela de login do usuário.
- **userAlimentosConsumidos**: Tela que exibe os alimentos consumidos pelo usuário.

Cada uma dessas telas é implementada em arquivos `.tsx`, permitindo a utilização de JSX para a construção da interface do usuário. A estrutura modular facilita a manutenção e a escalabilidade da aplicação.
